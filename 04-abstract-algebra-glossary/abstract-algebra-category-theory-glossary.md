## Chapter 1: Foundations of Category Theory

**Category.** A category C consists of a collection of objects and, for each pair of objects X and Y, a collection of morphisms (or arrows) from X to Y. Every object has an identity morphism. Morphisms compose associatively. The identity morphism acts as a unit for composition. Categories range from the very concrete — sets and functions — to the very abstract — formal languages and deductions.

**Object.** An element of the collection ob(C). Objects may be sets, groups, spaces, logical propositions, or any mathematical entity. In category theory, the internal structure of objects is secondary; what matters is how they relate to other objects via morphisms.

**Morphism (Arrow).** A morphism f from X to Y, written f: X → Y, is an element of Hom(X, Y). The object X is the domain (source) and Y is the codomain (target). Morphisms generalize functions, homomorphisms, continuous maps, and logical proofs.

**Composition.** Given morphisms f: X → Y and g: Y → Z, their composite g ∘ f: X → Z is defined. Composition is associative: h ∘ (g ∘ f) = (h ∘ g) ∘ f. The notation g ∘ f reads "g after f," reflecting the order of application.

**Identity morphism.** For each object X, there is a morphism id_X: X → X. It satisfies f ∘ id_X = f and id_Y ∘ f = f for any f: X → Y. Identity morphisms are unique.

**Small category.** A category where the collection of objects and all hom-collections are sets (not proper classes). The category of finite sets and the category of finitely generated groups are small. The category of all sets is not small.

**Large category.** A category whose objects or morphisms form a proper class. The category Set of all sets is large. Most categories of interest in mathematics are large.

**Locally small category.** A large category where each hom-collection Hom(X, Y) is a set. This is the typical assumption. Set, Grp, Top, and Ring are all locally small.

**Discrete category.** A category where the only morphisms are identity morphisms. The objects form a set with no relationships between distinct objects.

**Indiscrete category.** A category where there is exactly one morphism between any two objects (in each direction). Maximally connected.

**Opposite category.** The category C^op obtained by reversing all arrows in C. Objects are the same; the morphism f: X → Y in C becomes f: Y → X in C^op. Duality — the simultaneous replacement of C with C^op — is a fundamental principle.

## Chapter 2: Functors

**Functor.** A functor F: C → D is a map between categories preserving structure. It assigns to each object X in C an object F(X) in D, and to each morphism f: X → Y a morphism F(f): F(X) → F(Y), satisfying F(id_X) = id_{F(X)} and F(g ∘ f) = F(g) ∘ F(f).

**Covariant functor.** A functor in the standard sense: it preserves the direction of morphisms.

**Contravariant functor.** A functor that reverses morphisms: f: X → Y maps to F(f): F(Y) → F(X). Equivalently, a covariant functor from C^op. Cohomology is the canonical example.

**Forgetful functor.** A functor U: Grp → Set that "forgets" structure, sending each group to its underlying set and each homomorphism to the underlying function. Forgetful functors exist for virtually every algebraic category.

**Free functor.** Left adjoint to the forgetful functor. The free group functor F: Set → Grp sends each set S to the free group generated by S. Any function from S to the underlying set of a group G extends uniquely to a group homomorphism from F(S) to G.

**Full functor.** A functor F: C → D is full if for every pair of objects X, Y in C, the function Hom_C(X, Y) → Hom_D(F(X), F(Y)) induced by F is surjective.

**Faithful functor.** A functor F is faithful if the induced function on hom-sets is injective.

**Fully faithful functor.** Both full and faithful. A fully faithful functor is an embedding: it identifies C with a subcategory of D without identifying distinct objects or morphisms.

**Essentially surjective functor.** A functor F: C → D is essentially surjective if every object in D is isomorphic to F(X) for some object X in C.

**Equivalence of categories.** A functor F: C → D that is full, faithful, and essentially surjective. Equivalence is the correct notion of sameness for categories.

**Endofunctor.** A functor from a category to itself: F: C → C.

**Identity functor.** The functor id_C: C → C sending every object and morphism to itself.

**Constant functor.** The functor Δ_X: J → C sending every object in J to X and every morphism to id_X.

**Hom-functor.** For a fixed object A, the functor Hom(A, -): C → Set sending X to Hom(A, X) and f: X → Y to the function post-composing with f.

**Representable functor.** A functor F: C → Set is representable if it is naturally isomorphic to Hom(A, -) for some object A, called the representing object.

## Chapter 3: Natural Transformations

**Natural transformation.** Given functors F, G: C → D, a natural transformation η: F ⇒ G assigns to each object X in C a morphism η_X: F(X) → G(X) in D, such that for every f: X → Y, the naturality square commutes: G(f) ∘ η_X = η_Y ∘ F(f).

**Component.** Each morphism η_X is a component of the natural transformation η.

**Naturality square.** The commutative square asserting that the components of a natural transformation are compatible with all morphisms in the source category.

**Natural isomorphism.** A natural transformation η where every component η_X is an isomorphism. Two functors related by a natural isomorphism are called naturally isomorphic.

**Functor category.** The category [C, D] (or D^C) whose objects are functors C → D and whose morphisms are natural transformations. Composition of natural transformations is defined componentwise.

**Vertical composition.** Given natural transformations η: F ⇒ G and ε: G ⇒ H, their vertical composite ε ∘ η: F ⇒ H has components (ε ∘ η)_X = ε_X ∘ η_X.

**Horizontal composition (Godement product).** Given natural transformations η: F ⇒ G (between functors C → D) and ε: H ⇒ K (between functors D → E), their horizontal composite ε ★ η: H∘F ⇒ K∘G is defined using the naturality of the components.

**Interchange law.** Horizontal and vertical composition satisfy (ε' ∘ ε) ★ (η' ∘ η) = (ε' ★ η') ∘ (ε ★ η). This is the fundamental coherence law of 2-categories.

## Chapter 4: Universal Properties and Limits

**Universal property.** A universal property characterizes an object by specifying a unique factorization through it. Objects satisfying a universal property are unique up to unique isomorphism.

**Terminal object.** An object 1 such that for every object X, there is exactly one morphism X → 1. In Set, any singleton is terminal. In a category with a terminal object, it is unique up to unique isomorphism.

**Initial object.** Dual to terminal: an object 0 such that for every X, there is exactly one morphism 0 → X. In Set, the empty set is initial.

**Zero object.** An object that is both initial and terminal. Exists in Ab, R-Mod, and any abelian category.

**Cone.** For a diagram F: J → C, a cone is an object X (the apex) with morphisms λ_j: X → F(j) commuting with the diagram structure.

**Limit.** A universal cone: a cone (lim F, {λ_j}) such that any other cone (X, {μ_j}) factors uniquely through it via a unique morphism X → lim F.

**Colimit.** Dual to limit: a universal cocone. A cocone has morphisms from F(j) to the nadir, and the colimit has a unique map from it to any other cocone apex.

**Product.** Limit over the discrete two-object diagram. Comes with projection morphisms π_1: A×B → A and π_2: A×B → B.

**Coproduct.** Colimit over the discrete two-object diagram. Comes with injection morphisms ι_1: A → A⊔B and ι_2: B → A⊔B.

**Equalizer.** Limit of two parallel morphisms f, g: A → B. The equalizer eq(f,g) is the largest subobject of A on which f and g agree.

**Coequalizer.** Colimit of two parallel morphisms. The coequalizer coeq(f,g) is the quotient of B forcing f and g to become equal.

**Pullback.** Limit of a span A → C ← B. Consists of an object with morphisms to A and B making the square commute. In Set, the pullback is {(a,b) : f(a) = g(b)}.

**Pushout.** Colimit of a cospan A ← C → B. Dual to pullback.

**Complete category.** A category with all small limits.

**Cocomplete category.** A category with all small colimits.

## Chapter 5: The Yoneda Lemma

**Yoneda Lemma.** For any functor F: C → Set and object A in C:
Nat(Hom(A, -), F) ≅ F(A)
The bijection is natural in both F and A.

**Yoneda embedding.** The functor よ: C → [C^op, Set] sending A to Hom(-, A). By the Yoneda lemma, this is full and faithful.

**Density theorem.** Every presheaf (functor C^op → Set) is a canonical colimit of representable presheaves. This is the non-abelian analogue of the fact that every module is a colimit of free modules.

**Representability.** A functor F: C → Set is representable iff it has a universal element: a pair (A, u ∈ F(A)) such that for every (X, x ∈ F(X)), there is a unique f: A → X with F(f)(u) = x.

**Presheaf category.** The functor category [C^op, Set]. Contains C via the Yoneda embedding and is complete and cocomplete.

## Chapter 6: Adjunctions

**Adjunction.** Functors F: C → D and G: D → C are adjoint (F ⊣ G) if there is a natural bijection Hom_D(F(X), Y) ≅ Hom_C(X, G(Y)).

**Unit.** The natural transformation η: id_C ⇒ G∘F. The component η_X: X → G(F(X)) is the "insertion of generators."

**Counit.** The natural transformation ε: F∘G ⇒ id_D. The component ε_Y: F(G(Y)) → Y is the "evaluation map."

**Triangle identities.** (G ε) ∘ (η G) = id_G and (ε F) ∘ (F η) = id_F. These identities characterize the adjunction in terms of unit and counit.

**Adjoint functor theorems.** Freyd's theorem: G has a left adjoint iff it preserves all limits and satisfies the solution set condition. The special adjoint functor theorem gives simpler conditions for well-powered complete categories.

**Galois connection.** An adjunction between posets (viewed as categories). Equivalently, a pair of order-reversing maps satisfying a ≤ G(b) iff F(a) ≤ b. The classical Galois correspondence between field extensions and Galois groups is an instance.

**Reflective subcategory.** A full subcategory D ⊆ C such that the inclusion has a left adjoint (the reflector). Examples: abelian groups as a reflective subcategory of groups (abelianization); compact Hausdorff spaces as a reflective subcategory of topological spaces (Stone-Čech compactification).

## Chapter 7: Monads

**Monad.** A triple (T, η, μ) where T: C → C is a functor, η: id_C ⇒ T is the unit, and μ: T² ⇒ T is the multiplication, satisfying: μ ∘ T(μ) = μ ∘ μ_T (associativity) and μ ∘ T(η) = id_T = μ ∘ η_T (unit laws).

**Comonad.** Dual to a monad: (W, ε, δ) where ε: W ⇒ id is the counit and δ: W ⇒ W² is the comultiplication.

**T-algebra (Eilenberg-Moore algebra).** A pair (X, h: T(X) → X) where X is an object and h is a morphism satisfying h ∘ η_X = id_X and h ∘ T(h) = h ∘ μ_X.

**Eilenberg-Moore category.** The category T-Alg of T-algebras and their morphisms (maps commuting with the structure maps). The forgetful functor T-Alg → C has left adjoint, and the resulting monad is T.

**Kleisli category.** The category C_T whose objects are those of C, and where a morphism X → Y is a morphism X → T(Y) in C. Composition uses the multiplication μ.

**Beck's monadicity theorem.** G: D → C is monadic iff G has a left adjoint, G reflects isomorphisms, and D has coequalizers of G-split pairs that G preserves.

**Coalgebra.** An algebra for a comonad. A pair (X, δ: X → W(X)) satisfying counit and coassociativity axioms.

## Chapter 8: Abelian Categories and Homological Algebra

**Additive category.** A category where each hom-set is an abelian group and composition is bilinear. Has a zero object and all finite biproducts.

**Abelian category.** An additive category where every morphism has a kernel and cokernel, every monomorphism is a kernel, and every epimorphism is a cokernel.

**Exact sequence.** A sequence ⋯ → A → B → C → ⋯ where the image of each map equals the kernel of the next.

**Short exact sequence.** 0 → A → B → C → 0. This says A is a subobject of B with quotient C.

**Split exact sequence.** A short exact sequence that is isomorphic to 0 → A → A⊕C → C → 0.

**Snake lemma.** A diagram-chasing result: given a commutative diagram with exact rows, there is an exact sequence connecting the kernels and cokernels. Foundational for homological algebra.

**Chain complex.** A sequence C_n with boundary maps d_n: C_n → C_{n-1} satisfying d_{n-1} ∘ d_n = 0.

**Homology.** H_n(C) = ker(d_n) / im(d_{n+1}). Measures the failure of exactness.

**Cochain complex and cohomology.** The dual: C^n with coboundary maps d^n: C^n → C^{n+1}. Cohomology H^n = ker(d^n) / im(d^{n-1}).

**Projective resolution.** A resolution ⋯ → P_1 → P_0 → M → 0 where each P_i is projective (a direct summand of a free object). Used to compute left-derived functors.

**Injective resolution.** The dual: 0 → M → I^0 → I^1 → ⋯ with injective objects. Used to compute right-derived functors.

**Derived functor.** For a right-exact functor F, the left-derived functors L_n F measure the failure of exactness. Computed via projective resolutions.

**Ext functor.** Ext^n(A, B) = n-th right derived functor of Hom(A, -). Classifies n-fold extensions of B by A.

**Tor functor.** Tor_n(A, B) = n-th left derived functor of A ⊗ -. Measures the failure of flatness.

**Freyd-Mitchell embedding theorem.** Every small abelian category embeds fully faithfully and exactly into R-Mod for some ring R.

## Chapter 9: Topos Theory and Higher Structures

**Topos (elementary).** A category that is Cartesian closed and has a subobject classifier. Behaves like a universe of sets. Every topos has finite limits, a subobject classifier, and exponentials.

**Cartesian closed category.** A category with all finite products and, for each A, a right adjoint to the product functor (-) × A, called the exponential (-)^A.

**Subobject classifier.** An object Ω with a morphism true: 1 → Ω such that every monomorphism S → X is a pullback of true along a unique characteristic morphism X → Ω.

**Grothendieck topos.** A category of sheaves on a site. Always an elementary topos. Has a generating set and is locally presentable.

**Site.** A category with a Grothendieck topology: a specification of "covering sieves" satisfying stability and transitivity axioms.

**Sheaf.** A functor F: C^op → Set on a site satisfying the gluing condition: sections that agree on overlaps glue uniquely.

**Geometric morphism.** A functor f_*: E → F between toposes with a left exact left adjoint f*: F → E.

**Enriched category.** A category where hom-sets are replaced by hom-objects in a monoidal category V. The composition and identity are morphisms in V rather than functions.

**Monoidal category.** A category with a tensor product ⊗ and unit object I, with associativity and unit isomorphisms satisfying coherence axioms.

**Symmetric monoidal category.** A monoidal category with a braiding isomorphism γ_{A,B}: A⊗B → B⊗A satisfying symmetry and coherence conditions.

**2-Category.** A category enriched over Cat. Has objects (0-cells), morphisms (1-cells), and morphisms between morphisms (2-cells). Cat itself is a 2-category.

**Bicategory.** A weakened 2-category where associativity and unit laws hold only up to coherent isomorphism (not strictly).

**Infinity-category (quasi-category).** A simplicial set satisfying the inner horn filling condition. Composition defined up to coherent higher homotopy. The framework for homotopy-coherent mathematics.

**Kan extension.** The universal way to extend a functor along another. Left Kan extension is characterized by: Nat(Lan_K F, G) ≅ Nat(F, G∘K). Mac Lane: "All concepts are Kan extensions."

**Fibered category (Grothendieck fibration).** A functor p: E → B where every morphism in B can be lifted to a Cartesian morphism in E. Encodes a family of categories parameterized by B.

## Chapter 10: Key Theorems and Principles

**Duality principle.** Any theorem about a category C yields a dual theorem about C^op by reversing all arrows. Limits dualize to colimits; monomorphisms dualize to epimorphisms; initial to terminal.

**Density theorem.** Every presheaf is a colimit of representables: F ≅ colim_{(A, x) ∈ ∫F} Hom(-, A).

**Freyd-Mitchell embedding theorem.** Every small abelian category fully faithfully embeds into a module category, preserving all exact sequences.

**Beck's monadicity theorem.** Characterizes when a functor is monadic in terms of reflecting isomorphisms and handling certain coequalizers.

**Barr-Beck theorem.** A more general version of Beck's theorem applicable to comonadic descent.

**Mac Lane coherence theorem.** Every monoidal category is monoidally equivalent to a strict monoidal category. Similarly for symmetric monoidal categories.

**Eckmann-Hilton argument.** If a set has two monoid structures where each distributes over the other and they share a unit, then the two operations coincide and are commutative. Explains why homotopy groups π_n for n ≥ 2 are abelian.

**Adjoint functor theorem (Freyd).** G: D → C has a left adjoint iff G preserves all limits and the solution set condition holds.

**Abstract nonsense.** A proof by general abstract nonsense derives its conclusions from universal properties, functoriality, and naturality rather than element-wise arguments. While the term is humorous, it reflects the genuine power of category theory: many important results can be proved at this level of generality, yielding theorems that apply uniformly across different mathematical domains.

## Conclusion

Category theory occupies a unique position in mathematics. It is simultaneously a foundational framework — a language in which all of mathematics can be expressed; a unifying theory — revealing deep connections between algebra, topology, logic, and computation; and a practical tool — offering powerful techniques for constructing and analyzing mathematical objects.

The beauty of category theory lies in its principle that mathematical objects are best understood not by examining their internal constituents but by studying the morphisms between them. This relational perspective has proven extraordinarily fruitful, and the vocabulary introduced in this glossary provides the tools needed to participate in this ongoing mathematical conversation.
