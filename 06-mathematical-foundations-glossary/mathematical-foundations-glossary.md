## Part I: Abstract Algebra

### Groups

**Group.** A set G with a binary operation · satisfying: closure (a·b ∈ G), associativity ((a·b)·c = a·(b·c)), identity (∃e: e·a = a·e = a), and inverses (∀a ∃a⁻¹: a·a⁻¹ = a⁻¹·a = e). Groups are the fundamental algebraic structure capturing symmetry.

**Abelian (commutative) group.** A group where a·b = b·a for all a, b.

**Subgroup.** A subset H ≤ G that is a group under the same operation. Criterion: H is non-empty, and for all a, b ∈ H: a·b⁻¹ ∈ H.

**Normal subgroup.** N ⊴ G if gNg⁻¹ = N for all g ∈ G. Normal subgroups are kernels of homomorphisms and allow construction of quotient groups.

**Quotient group.** G/N: the group of cosets gN under the operation (gN)(hN) = (gh)N. Well-defined iff N is normal.

**Coset.** Left coset: gH = {gh : h ∈ H}. Right coset: Hg. In general gH ≠ Hg.

**Group homomorphism.** A function φ: G → H with φ(ab) = φ(a)φ(b) for all a, b ∈ G.

**Kernel.** ker(φ) = {g ∈ G : φ(g) = e_H}. Always a normal subgroup of G.

**Image.** im(φ) = {φ(g) : g ∈ G}. A subgroup of H.

**Isomorphism.** A bijective homomorphism. G ≅ H means G and H are "the same group."

**Automorphism.** An isomorphism from G to itself. Aut(G) is itself a group.

**Group action.** A homomorphism G → Sym(X) for some set X. Equivalently: a function G × X → X, (g,x) ↦ g·x, satisfying e·x = x and g·(h·x) = (gh)·x.

**Lagrange's theorem.** If H ≤ G and G is finite, then |H| divides |G|. The index [G:H] = |G|/|H|.

**Sylow theorems.** For a finite group G of order p^n · m (p prime, p∤m):
- Sylow p-subgroups of order p^n exist
- All Sylow p-subgroups are conjugate
- The number of Sylow p-subgroups divides m and is congruent to 1 mod p

**Cyclic group.** A group generated by one element: G = ⟨g⟩ = {g^n : n ∈ ℤ}. The cyclic group of order n is isomorphic to ℤ/nℤ.

**Simple group.** A non-trivial group with no proper normal subgroups. The building blocks of all finite groups (Jordan-Hölder theorem).

**Free group.** F(S): the group generated by a set S with no relations except those forced by the group axioms. Every element is a reduced word in the generators and their inverses.

### Rings

**Ring.** A set R with two operations + and · where (R, +) is an abelian group, · is associative, and · distributes over + from both sides.

**Commutative ring.** A ring where a·b = b·a for all a, b.

**Ring with unity.** A ring with a multiplicative identity element 1 ≠ 0.

**Unit.** An element u ∈ R with a multiplicative inverse u⁻¹ ∈ R.

**Ideal.** A subset I ⊆ R satisfying: (I, +) is a subgroup, and for all r ∈ R and i ∈ I: r·i ∈ I and i·r ∈ I. Ideals are kernels of ring homomorphisms.

**Principal ideal.** The ideal generated by one element: (a) = {ra : r ∈ R} in a commutative ring.

**Prime ideal.** An ideal P such that if ab ∈ P then a ∈ P or b ∈ P. Equivalently, R/P is an integral domain.

**Maximal ideal.** An ideal M ≠ R such that no ideal I satisfies M ⊊ I ⊊ R. Equivalently, R/M is a field.

**Ring homomorphism.** φ: R → S with φ(a+b) = φ(a)+φ(b) and φ(ab) = φ(a)φ(b).

**Quotient ring.** R/I: the ring of cosets r+I under (r+I)+(s+I) = (r+s)+I and (r+I)·(s+I) = rs+I.

**Integral domain.** A commutative ring with unity and no zero divisors (ab = 0 implies a = 0 or b = 0).

**Principal ideal domain (PID).** An integral domain where every ideal is principal. Examples: ℤ, F[x] for any field F.

**Unique factorization domain (UFD).** An integral domain where every non-zero non-unit factors uniquely into irreducibles (up to order and units).

**Euclidean domain.** An integral domain with a Euclidean algorithm. Euclidean → PID → UFD.

**Polynomial ring.** R[x]: polynomials with coefficients in R. If R is a UFD, so is R[x] (Gauss's lemma).

### Fields and Galois Theory

**Field.** A commutative ring with unity where every non-zero element is a unit.

**Characteristic.** The smallest positive n such that n·1 = 0, or 0 if no such n exists.

**Field extension.** K/F (K contains F as a subfield).

**Degree [K:F].** The dimension of K as an F-vector space.

**Algebraic element.** α ∈ K is algebraic over F if it satisfies a polynomial with F-coefficients.

**Transcendental element.** Not algebraic.

**Splitting field.** The smallest extension of F over which a polynomial factors completely into linear factors.

**Galois group.** Gal(K/F) = Aut_F(K): the group of field automorphisms of K fixing F pointwise.

**Fundamental theorem of Galois theory.** For a Galois extension K/F, there is an order-reversing bijection between intermediate fields and subgroups of Gal(K/F): E ↔ Gal(K/E).

**Solvable by radicals.** A polynomial is solvable by radicals iff its Galois group is solvable (has a composition series with abelian factors). General polynomials of degree ≥ 5 are not solvable — there is no quintic formula.

### Modules

**Module.** An abelian group M with a ring action: R × M → M satisfying r(m+n) = rm+rn, (r+s)m = rm+sm, (rs)m = r(sm), 1m = m.

**Free module.** A module isomorphic to R^n (direct sum of n copies of R).

**Projective module.** A direct summand of a free module. Equivalently: the Hom(P,-) functor is exact.

**Injective module.** I is injective if the Hom(-,I) functor is exact. Equivalently: every diagram 0 → A → B can be extended along any injection A → I.

**Flat module.** M is flat if M ⊗_R - is exact.

## Part II: Category Theory

*(See the companion Category Theory Glossary for the full treatment. Key terms summarized here.)*

**Category.** Objects and morphisms with identity and associative composition.

**Functor.** Structure-preserving map between categories.

**Natural transformation.** Morphism between functors.

**Adjoint functors.** F ⊣ G iff Hom(F(X), Y) ≅ Hom(X, G(Y)) naturally.

**Limit / Colimit.** Universal cone / cocone over a diagram.

**Yoneda Lemma.** Nat(Hom(A,-), F) ≅ F(A). Objects are determined by their relationships.

**Monad.** Endofunctor with unit and multiplication satisfying monad laws.

**Topos.** Cartesian closed category with a subobject classifier.

**Subobject classifier.** Object Ω with true: 1 → Ω classifying subobjects.

**Kan extension.** Universal way to extend a functor along another.

## Part III: ZFC Set Theory

### Basic Set Operations

**Union.** A ∪ B = {x : x ∈ A or x ∈ B}.

**Intersection.** A ∩ B = {x : x ∈ A and x ∈ B}.

**Complement.** A \ B = {x ∈ A : x ∉ B}. Relative complement.

**Power set.** P(A) = {S : S ⊆ A}. For finite A with |A| = n, |P(A)| = 2^n.

**Cartesian product.** A × B = {(a,b) : a ∈ A, b ∈ B}. Ordered pairs defined as: (a,b) = {{a}, {a,b}}.

### The Nine ZFC Axioms

**Axiom of Extensionality.** ∀x∀y(∀z(z∈x ↔ z∈y) → x=y). Sets are equal iff they have the same elements.

**Axiom of Empty Set.** ∃x∀y(y∉x). The empty set ∅ exists.

**Axiom of Pairing.** ∀x∀y∃z∀w(w∈z ↔ w=x ∨ w=y). For any x, y, the set {x, y} exists.

**Axiom of Union.** ∀F∃A∀x(x∈A ↔ ∃B(x∈B ∧ B∈F)). The union of a family of sets is a set.

**Axiom of Power Set.** ∀x∃P∀z(z∈P ↔ z⊆x). The power set exists.

**Axiom of Infinity.** ∃S(∅∈S ∧ ∀x∈S(x∪{x}∈S)). There is an infinite set.

**Axiom Schema of Replacement.** If F is a function-class (definable by a formula), then for any set A, F(A) = {F(x) : x ∈ A} is a set.

**Axiom of Regularity (Foundation).** ∀x(x≠∅ → ∃y∈x(y∩x=∅)). Every non-empty set has an ∈-minimal element. Rules out x∈x and infinite descending chains.

**Axiom Schema of Separation.** ∀A∀(parameter)∃B∀x(x∈B ↔ x∈A ∧ φ(x)). Subsets defined by formulas exist.

### The Axiom of Choice

**Axiom of Choice (AC).** For every family of non-empty sets {A_i : i∈I}, there exists a choice function f with f(i) ∈ A_i for all i.

**Zorn's Lemma.** Every partially ordered set in which every chain has an upper bound has a maximal element. Equivalent to AC.

**Well-Ordering Theorem.** Every set can be well-ordered. Equivalent to AC.

**Hausdorff Maximal Principle.** Every chain in a partially ordered set extends to a maximal chain. Equivalent to AC.

**Independence of AC.** Gödel proved AC is consistent with ZF (it holds in the constructible universe L). Cohen proved ¬AC is consistent with ZF (via forcing). AC is independent.

### Ordinals and Cardinals

**Ordinal.** A transitive set (every element is a subset) well-ordered by ∈. The von Neumann ordinals: 0=∅, 1={0}, 2={0,1}, ω={0,1,2,...}, ω+1={0,1,...,ω}, ...

**Successor ordinal.** α+1 = α∪{α}.

**Limit ordinal.** An ordinal that is not 0 and not a successor. ω, ω², ε₀ are limit ordinals.

**Transfinite induction.** To prove P(α) for all ordinals: prove P(0); prove P(α)→P(α+1); prove (∀β<λ P(β))→P(λ) for limit ordinals λ.

**Transfinite recursion.** Define f(α) using f(β) for all β<α.

**Cardinal.** An ordinal not bijectable with any smaller ordinal.

**Aleph numbers.** ℵ₀ = ω (countably infinite), ℵ₁ = the first uncountable cardinal, ℵ_{α+1} = the successor cardinal to ℵ_α.

**Beth numbers.** ℶ₀ = ℵ₀, ℶ_{α+1} = 2^{ℶ_α}, ℶ_λ = sup_{α<λ} ℶ_α.

**Cofinality.** cf(α) = the smallest ordinal β such that a β-sequence is cofinal in α. A regular cardinal κ satisfies cf(κ) = κ.

**Cantor's diagonal argument.** For any set X, there is no surjection X → P(X). Proof: given f: X → P(X), the set D = {x ∈ X : x ∉ f(x)} satisfies D ∉ im(f).

### Set-Theoretic Constructions

**Functions.** A function f: A → B is a set of ordered pairs such that every a ∈ A appears in exactly one pair (a, b).

**Relations.** A subset R ⊆ A × B.

**Equivalence relation.** R is an equivalence relation on A if it is reflexive (aRa), symmetric (aRb → bRa), and transitive (aRb ∧ bRc → aRc).

**Equivalence class.** [a]_R = {b ∈ A : aRb}. Equivalence classes partition A.

**Quotient.** A/R = {[a]_R : a ∈ A}: the set of equivalence classes.

### The Cumulative Hierarchy

**Von Neumann universe V.** V = ∪_α V_α where V_0 = ∅, V_{α+1} = P(V_α), V_λ = ∪_{α<λ} V_α for limit ordinals.

Every set appears in some V_α. The rank of a set x is the smallest α with x ∈ V_{α+1}.

**Constructible universe L.** L = ∪_α L_α where L_0 = ∅, L_{α+1} = Def(L_α) (sets definable from L_α by first-order formulas with parameters), L_λ = ∪_{α<λ} L_α. In L, AC and GCH both hold.
